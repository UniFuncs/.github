# UniFuncs是什么？

UniFuncs是一家全球领先的深度搜索服务提供商。

# 产品介绍

官方网站：[unifuncs.com](https://unifuncs.com)

U深搜（[s.unifuncs.com](https://s.unifuncs.com)），主打深度搜索，被广泛用于信息挖掘、事实核查、背景调查等。
U深研（[dr.unifuncs.com](https://dr.unifuncs.com)），主打深度研究，被广泛用于市场调研、报告撰写、事件分析等。

Web Search API（[https://unifuncs.com/api#web-search](https://unifuncs.com/api#web-search)），强大的海内外实时搜索API，拥有丰富的信源。
Web Reader API（[https://unifuncs.com/api#web-reader](https://unifuncs.com/api#web-reader)），强大的网页抓取阅读API，稳定抓取目标网页。
Deep Search API（[https://unifuncs.com/api#deepsearch](https://unifuncs.com/api#deepsearch)），获得与U深搜同级别深度搜索能力的API，目前已推出旗舰深搜模型S2。
Deep Research API（[https://unifuncs.com/api#deepresearch](https://unifuncs.com/api#deepresearch)），获得与U深研同级别深度研究能力的API，目前已推出旗舰深搜模型U1-Pro。

UniFuncs SKILL集成：[https://github.com/UniFuncs/unifuncs-skill](https://github.com/UniFuncs/unifuncs-skill)
